# 拾句 · 灵感捕手

让每一次灵光乍现，都有处安放。
一款在浏览中快速收藏信息的轻量工具。

## 产品定位

- 专注浏览过程中的“即取即存”，降低记录成本
- 构建个人灵感碎片库，回顾与整理更高效

## 适用场景

- 学生：论文阅读中的关键结论，集中回顾
- 创作者：金句与配图素材，保留来源随用随取
- 职场：会议资料的关键数据点，统一管理
- 终身学习者：随手记录冷知识与片段，不再分散

## 解决痛点

- 复制粘贴步骤多，打断阅读与思考
- 截图分散，后续检索困难
- 书签仅存链接，丢失上下文信息

## 核心体验

- 文本：选中即存，自动记录来源与时间
- 图片：右键保存，保留说明与链接
- 链接：保存目标地址，支持回到原文位置
- 快照：一键页面截图（可视区域），便于回看

## 自动记录

- 网页标题、URL、时间戳
- 选中片段的上下文（前后若干字）
- 简要来源信息，帮助快速定位原文

## 管理与检索

- 卡片式管理，突出内容主体
- 关键词搜索覆盖内容与标题
- 来源与时间维度筛选，聚焦重点
- 按类型筛选，精准定位文本/图片/链接/快照

## 导出与分享

- 导出 ZIP，文档中引用图片的相对路径
- 图片资源一并打包，跨设备查看更稳定
- 保留来源信息，兼顾可读与可溯

## 隐私与安全

- 数据默认仅存本地，不上传服务器
- 权限透明、最小化请求
- 不做广告或画像，尊重数据主权

## 设计原则

- F 型动线，核心操作置顶易触达
- 轻盈风格与细节动效，提升体验
- 少而精，避免管理负担

## 上手与反馈

- 安装后提供简要引导与示例
- 默认设置亲和直观，可按需调整
- 开源地址：https://github.com/minorcell/pick-quote，欢迎反馈与贡献

## 愿景

- 让灵感片段得以沉淀与重访
- 让知识收藏变得轻松、可用、可信
- 与用户共同打磨细节，构建属于你的灵感库
