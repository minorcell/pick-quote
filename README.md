# 拾句 · Pick Quote

> 让每一次灵光乍现，都有处安放

一款轻量级浏览器插件，在浏览网页时快速收藏灵感片段。支持文本、图片、链接和页面快照，自动保留上下文信息，所有数据仅存储在本地。

![拾句插件截图](https://github.com/user-attachments/assets/0b9dfd52-eb77-4ff2-a618-5236f33b0b36)

## 核心功能

### 即取即存

- **文本收藏**：选中文字，右键一键保存，自动记录来源与时间
- **图片保存**：右键保存图片，保留说明与原始链接
- **链接收藏**：保存目标地址，支持回到原文位置
- **页面快照**：一键截图当前可视区域，便于回看

### 智能记录

- 自动捕获网页标题、URL、时间戳
- 保存选中内容的上下文（前后文段落）
- 记录简要来源信息，帮助快速定位原文
- 智能去重，避免重复保存相同内容

### 优雅管理

- 瀑布流式展示，视觉体验舒适
- 支持亮/暗双主题，自动跟随系统
- 文艺风格界面，衬线字体营造阅读感
- 响应式设计，完美适配移动端

## 适用场景

### 学生群体

论文阅读中的关键结论、重要引用，集中回顾更高效

### 内容创作者

金句收集、配图素材整理，保留来源随用随取

### 职场人士

会议资料的关键数据点、重要信息，统一管理不遗漏

### 终身学习者

随手记录冷知识、有趣片段，不再分散各处

## 隐私安全

- **本地存储**：所有数据仅存本地 IndexedDB，不上传服务器
- **权限透明**：最小化权限请求，仅必要的浏览器权限
- **无广告追踪**：不做广告或用户画像，尊重数据主权
- **开源透明**：代码完全开源，可审计验证

## 导出分享

- **ZIP打包**：将所有收藏导出为 Markdown，图片嵌入文档
- **卡片分享**：单个条目可导出为精美图片卡片，适合社交媒体分享
- **来源保留**：导出内容保留完整来源信息，兼顾可读与可溯

## 开发部署

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 生产构建
pnpm build

# 打包扩展
pnpm package
```

## 设计特色

采用文艺风格设计哲学：

- **温暖配色**：米白背景配深色文字，营造舒适阅读体验
- **优雅字体**：Noto Serif SC 衬线字体，提升内容质感
- **柔和圆角**：12px 圆角边框，视觉更加友好
- **高可读性**：1.8 倍行高，长时间阅读不疲劳

## 多端适配

- **桌面端**：三栏瀑布流，充分利用屏幕空间
- **平板端**：双栏布局，平衡信息密度与可读性
- **手机端**：单栏设计，专注内容浏览体验
